# YellowMarket — Stratégie de Ciblage Industries (Footer & Navigation)

## 1. État des lieux

**Footer actuel** (`.footer-seo-links`) :
- Services (redondant avec le nav footer principal) : AI Website Design, GBP, AI Lead Follow-Up
- Industries spécifiques : Plumbers, Roofers, HVAC, Electricians, Cleaning Services, Dentists

**Problème** : 6 industries = 6 niches. Ça exclut massivement : auto repair, landscapers, lawyers, real estate agents, restaurants, barbers, salons, clinics, pest control, painters… et des centaines d'autres.

**Impossible** de toutes les lister → le footer exploserait ou deviendrait du bruit.

## 3. RECOMMANDATION : Catégories dans le footer, pages spécifiques en SEO silencieux

### Verdict : PASSER AUX CATÉGORIES

| Approche | Pourquoi pas |
|----------|-------------|
| ❌ **Garder les 6 industries spécifiques** | Trop étroit, exclut 80% des prospects |
| ❌ **Lister 15+ industries** | Pas scalable, alourdit le footer, dilue le message |
| ❌ **Solution générique "all industries"** | Teo ne veut pas — manque de réflexion stratégique |
| ✅ **3-4 catégories dans le footer** | Couvre tout le TAM, propre, scalable, cohérent avec le "who we help" |

### Les 3 catégories recommandées

| Catégorie | URL | Couvre | Marché adressable |
|-----------|-----|--------|-------------------|
| **Home Services** | `/industries/home-services/` | Plumbers, Roofers, HVAC, Electricians, Landscapers, Cleaners, Pest Control, Painters | $620B |
| **Auto & Trades** | `/industries/auto-trades/` | Auto Repair, Body Shops, Mechanics, Tire Shops, Mobile Mechanics | $70B+ |
| **Professional Services** | `/industries/professional-services/` | Dentists, Lawyers, Real Estate, Clinics, Accountants, Consultants, Recruiters | $200B+ |

### Pourquoi les catégories gagnent

1. **Cohérence** : Le "who we help" de la homepage utilise déjà "Local service businesses" et "High-value service teams" — on aligne le footer avec ce langage.
2. **Breadth signal** : "Home Services" en dit plus long que "For Plumbers" + "For Roofers" + "For HVAC" — le prospect comprend instantanément que tu sers son industrie.
3. **Scalabilité SEO** : Chaque `/industries/home-services/` peut lier silencieusement vers des sous-pages spécifiques (plumbers, roofers, etc.) sans alourdir le footer.
4. **Moins de bruit** : 3 liens au lieu de 6. Chaque lien a plus de valeur.
5. **Match produit** : Les services YellowMarket (Website + GBP + Follow-Up) sont les mêmes pour toutes ces industries — pas besoin de les distinguer dans le footer.

## 4. Code exact pour remplacer `.footer-seo-links`

### Avant (lignes 268-280)
```html

        <span style="color:rgba(255,255,255,.35);font-size:.7rem;text-transform:uppercase;letter-spacing:.08em;font-weight:900">Services</span>
        <a href="/services/ai-website-design/">AI Website Design</a>
        <a href="/services/google-business-profile-optimization/">Google Business Profile</a>
        <a href="/services/ai-lead-follow-up/">AI Lead Follow-Up</a>
        <span style="color:rgba(255,255,255,.35);font-size:.7rem;text-transform:uppercase;letter-spacing:.08em;font-weight:900">Industries</span>
        <a href="/industries/plumbers/">For Plumbers</a>
        <a href="/industries/roofers/">For Roofers</a>
        <a href="/industries/hvac/">For HVAC</a>
        <a href="/industries/electricians/">For Electricians</a>
        <a href="/industries/cleaners/">For Cleaning Services</a>
        <a href="/industries/dentists/">For Dentists</a>
      </div>
```

### Après
```html
      <div class="footer-seo-links" style="width:100%;margin-top:18px;display:flex;gap:12px;flex-wrap:wrap;font-weight:800;font-size:.9rem;align-items:center">
        <span style="color:rgba(255,255,255,.35);font-size:.7rem;text-transform:uppercase;letter-spacing:.08em;font-weight:900">For You</span>
        <a href="/industries/home-services/">Home Services</a>
        <a href="/industries/auto-trades/">Auto &amp; Trades</a>
        <a href="/industries/professional-services/">Professional Services</a>
      </div>
```

### Ce qui change
- **Label** : `Industries` → `For You` (plus orienté prospect, moins catalogue)
- **Liens** : 6 spécifiques → 3 catégories
- **Coverage** : ~6 niches → tout le TAM local US
- **Redondance supprimée** : Le bloc "Services" (redondant avec le nav footer) est retiré
